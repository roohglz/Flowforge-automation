# Flowforge-automation
Cloud-based workflow automation platform built with Python, FastAPI, and Streamlit that automates document-intensive operations — extracts text, generates NLP summaries, and classifies documents across custom pipelines.

 FlowForge Automation Suite

A cloud-based workflow automation platform for document-intensive business operations.

 Features
- Create and manage automation workflows
- Upload PDF, CSV, TXT documents for auto-processing
- Automatic text extraction, NLP summarization, document classification
- Real-time analytics dashboard
- REST API with FastAPI (10 endpoints)

 🛠️ Tech Stack
- Python, FastAPI, Streamlit, SQLite, SQLAlchemy, Plotly

 Run Locally
pip install -r requirements.txt
streamlit run app.py
